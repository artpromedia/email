When printing/exporting:

1. **Remove UI chrome** - No buttons, dropdowns, or interactive elements
2. **Show full email** - All content visible
3. **Include metadata** - From, To, Subject, Date
4. **Preserve branding** - Keep header/footer HTML
5. **Clean formatting** - Remove hover states, focus rings
